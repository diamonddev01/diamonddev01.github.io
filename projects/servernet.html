Servernet

Servernet is a Discord guild to guild connection bot, now if that sounds complex dont worry I can explain it all. All Servernet does in your server os when someone sends a message in a certain channel the bot relays that message to other channels. The messages have filters and the bot is well moderated by a small but vigilant team of moderators. ServerNets main aim is to increase your servers activity using curiosity and the overall will to talk in an active place. We have over 20 servers on the main network with the bot still request only beta. We do plan on bringing it away from being a beta only bot soon. ServerNet website is servernets official website!
